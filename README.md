# Swift-hayashi-portfolio  
  
概要：iPhoneアプリゲーム（落ちモノゲーム）  
  
推奨環境：Mac、iPhone  
  
操作方法：全てタッチ（Macでは画面をマウスクリック）で操作します。  
　　　　　ゲームが開始したら、プレイヤー（ウーパールーパー）の左右をタッチしてプレイヤーを動かします。  

ゲームの目的：上から降ってくるエサにプレイヤーが触れると得点になります。  
　　　　　　　エサを取りこぼさないようにして、ゲームクリアの20点を目指します。  
　　　　　　　スコアが5点以下、6点以上19点、ゲームクリアの20点、の３パターンで、  
　　　　　　　スタート画面に戻った時の画像が変化します。  
    
　制作期間：1.5日  
  使用言語など：XCode15.4 Swift5.10  
  フレームワーク：SpriteKit  
  ※小規模な開発となっておりますゆえ、コミットは大きめの単位になっています。
